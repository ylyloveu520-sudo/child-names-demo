# 儿童识字小报生成器

一个纯前端单页应用，为5-9岁儿童自动生成识字小报图片。

## 功能特点

- **预置词库**：内置10个常见场景的识字词汇（超市、医院、公园、动物园、学校、图书馆、厨房、火车站、消防站、邮局）
- **AI图片生成**：使用Nano Banana Pro API生成精美的识字小报
- **历史记录**：自动保存生成历史，方便查看和下载
- **本地运行**：纯前端实现，无需后端服务器

## 使用方法

### 1. 获取API密钥

**Nano Banana Pro密钥**（用于图片生成）：
- 访问 [Kie AI API Key管理](https://kie.ai/api-key)
- 获取API密钥

### 2. 设置密钥

1. 打开应用，点击右上角"设置API密钥"
2. 输入API密钥
3. 点击"保存密钥"

### 3. 生成识字小报

1. 输入**主题/场景**（支持：超市、医院、公园、动物园、学校、图书馆、厨房、火车站、消防站、邮局）
2. 输入**小报标题**（如：走进超市、快乐医院）
3. 点击"开始生成小报"
4. 等待生成完成（通常需要30-60秒）

### 4. 下载图片

生成完成后，点击"下载图片"按钮保存到本地。

## 技术栈

- 纯HTML5 + CSS3 + Vanilla JavaScript
- 预置词库（词汇数据）
- Nano Banana Pro API（图片生成）
- localStorage（密钥和历史记录管理）

## 注意事项

1. API密钥存储在浏览器localStorage中，请勿在公共电脑上使用
2. 生成过程可能需要30-60秒，请耐心等待
3. 每次生成会消耗API调用额度
4. 建议在Chrome/Edge浏览器中使用

## 错误排查

**生成失败**：
- 检查API密钥是否正确
- 检查API账户是否有余额
- 查看浏览器控制台错误信息

**超时**：
- 图片生成需要较长时间，请耐心等待
- 如果超过5分钟未完成，请重试

**词汇不匹配**：
- 确保输入的场景名称在支持列表中
- 支持模糊匹配（如输入"水果店"会尝试匹配相似场景）
